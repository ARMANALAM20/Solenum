# Solenum

A Web3 project built on Solana blockchain.

## Description

Solenum is a decentralized application leveraging the Solana network for fast and efficient blockchain operations.

## Features

- Built on Solana blockchain
- Fast transaction processing
- Low transaction costs

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Solana CLI tools

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/Solenum.git

# Navigate to project directory
cd Solenum

# Install dependencies
npm install
```

### Usage

```bash
# Start the development server
npm run dev

# Build for production
npm run build
```
